Projects

Mars Rover Simulation - J2EE based project running on a server built with Node. Four teams worked in collaboration to share live information of their corresponding rover simulations via REST service to find science materials for a given map and build an AI for rover movement as well as data gathering.
React-Bookshelf - This React app allows users to select and categorize books that they have read, currently reading or want to read in the future.It has a feature for searching from a category running on the backend.
ScoreTracker Android - Android application providing real-time and live score updates for MLB, NBA. Has an embedded map that shows the users the route to a particular match venue from their current position.
Predicting House Sale Price - Predicts house sale prices in King County using Linear Regression, KNN, Decision Tree, Random Forest, Polynomial Regression and compares the results.Provides a comprehensive insight into the data and feature correlation using Scatter Plot, Heatmap and Principal Component Analysis
NPM Package Registration - Single Page Application - SPA built with AngularJS. Students will sign up with their desired package from NPM that they will present on. Users will be sorted in groups based on their selected package.
React-Youtube - This React application allows the user to search for a youtube channel/youtuber. The result will show the basic information for the channel and then show their top 10 playlists.
Online Store/Inventory - Online store for anonymous users and inventory management system for store admins
