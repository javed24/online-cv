Projects

Mars Rover Simulation - J2EE based project running on a server built with Node. Four teams worked in collaboration to share live information of their corresponding rover simulations via REST service to find science materials for a given map and build and AI for rover movement as well as data gathering.
ScoreTracker - Android application providing real-time and live score updates for MLB, NBA. Has an embedded map that shows the users the route to a particular match venue from their current position.
Spotify SPA - Single Page Application built with AngularJS that allows users to fetch the details for an artist or an album using the RESTful service  from the Spotify API.
Online Store/Inventory - Online store for anonymous users and inventory management system for store admins
Delta - A web application for travelers, who need a lodging service for their temporary stay in different locations away from home. Travelers can book  houses and owners can post ads with rental details.
